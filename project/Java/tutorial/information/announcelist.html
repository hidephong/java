Get Announcements about The Java Tutorial

The Java Tutorial announcements list is a low traffic e-mail list that is maintained and used only by the Java Tutorial team. We send announcements to this list when the online Tutorial is updated, new books are published, and so on. Only Java Tutorial team members can post to the list.

Sun Microsystems respects your desire for privacy. Data collected for this mailing list will not be given, sold, or shared with organizations external to Sun. This information will be used only to add you to a low-traffic mailing list that informs you of Tutorial updates. If you have any questions, refer to Sun's Privacy Policy.

How to Subscribe

To subscribe, send mail to 
java-tutorial-announce-request@eng.sun.com
and put the single word 
subscribe
in the body of the message.

If the subscription is successful, you will receive two e-mail messages. One is a short message that indicates you have been added to the list. The other is a longer message that contains a welcome message and instructions for unsubscribing, should you ever choose to do so.

How to Unsubscribe

Send mail to 
java-tutorial-announce-request@eng.sun.com
and put the single word 
unsubscribe
in the body of the message.

If you unsubscribe successfully, you will receive a single e-mail message that indicates you've been removed from the list.
